## Comandos iniciales
```
git clone https://github.com/ssaurexd/apil-login-client.git
cd api-login-client
npm install
```

## Variables ENV
```
NEXTAUTH_SECRET = 
NEXTAUTH_URL = 
NEXT_PUBLIC_JWT_SEED =
API_HOST = 
GOOGLE_SID =
GOOGLE_CLIENT_SECRET =
```

## Scripts dev
- Link: ``npm lint``
- Iniciar server: ``npm run dev``
